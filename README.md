# speech-247-latex-source

Pushes to `main` automatically compile `sn-article.tex` and publish the latest PDF to GitHub Pages.

After GitHub Pages is enabled for this repository, the manuscript should be available at:

- `https://bcm-neurosurgery.github.io/speech-247-latex-source/`
- `https://bcm-neurosurgery.github.io/speech-247-latex-source/paper.pdf`

The repository no longer needs to track the compiled manuscript PDF locally.
